# My Portfolio

Selamat datang di repositori portofolio saya! Website ini dibuat menggunakan React dan disusun untuk menampilkan berbagai karya, alat (tools), dan proyek-proyek saya.

## 🚀 Live Demo
Website ini dapat diakses secara langsung di: **[https://ilhamjaya.github.io/](https://ilhamjaya.github.io/)**

## 🛠️ Teknologi yang Digunakan
* **React** - Library utama untuk membangun antarmuka pengguna
* **Tailwind CSS** - Framework CSS untuk mendesain tampilan secara cepat
* **GitHub Pages** - Digunakan untuk proses hosting dan deployment secara otomatis via GitHub Actions

## ⚙️ Cara Menjalankan Secara Lokal
Jika Anda ingin menjalankan proyek ini di komputer Anda sendiri:

1. Clone repositori ini:
   ```bash
   git clone https://github.com/IlhamJaya/IlhamJaya.github.io.git
   ```
2. Pindah ke dalam folder proyek:
   ```bash
   cd IlhamJaya.github.io
   ```
3. Install semua *dependencies*:
   ```bash
   npm install
   ```
4. Jalankan server lokal (development):
   ```bash
   npm start
   ```
5. Buka `http://localhost:3000` di browser Anda!
